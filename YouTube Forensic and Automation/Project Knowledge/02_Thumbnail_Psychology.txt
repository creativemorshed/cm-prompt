Threat

3. Shock

THUMBNAIL COMPOSITION FRAMEWORK

Rule 1

One Primary Focus

Viewer should know where to look.

Bad:

Too many objects.

Good:

Single focus.

--------------------------------------------------

Rule 2

Visual Hierarchy

Most important object biggest.

--------------------------------------------------

Rule 3

Eye Direction

Humans follow eyes.

If person looks somewhere:

Viewer looks there.

--------------------------------------------------

Rule 4

Visual Tension

Create unanswered question.

Example:

Person shocked

But reason hidden.
